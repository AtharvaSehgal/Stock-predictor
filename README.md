# 📈 NSE Stock Predictor
AI-powered Indian stock analysis tool built on MacBook Air M4.

## Stack
- Data: yfinance (NSE/BSE live)
- ML Model: LSTM (TensorFlow)
- Sentiment: VADER + NewsAPI
- AI Insights: Groq (Llama3, free)
- Dashboard: Streamlit + Plotly

## Run locally
pip install -r requirements.txt
streamlit run app.py
